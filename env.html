Variables de entorno para Gmail

Usa estas variables en tu entorno local (.env.local) y en Vercel (Project Settings → Environment Variables). Elige una de las dos configuraciones.

Configuración recomendada (SSL 465)

SMTP_HOST=smtp.gmail.com
SMTP_PORT=465
SMTP_SECURE=true
SMTP_USER=josefranciscobarretolopez@gmail.com
SMTP_PASS=<TU_APP_PASSWORD>
MAIL_TO=josefranciscobarretolopez@gmail.com
MAIL_FROM=josefranciscobarretolopez@gmail.com

Configuración alternativa (STARTTLS 587)

SMTP_HOST=smtp.gmail.com
SMTP_PORT=587
SMTP_SECURE=false
SMTP_USER=josefranciscobarretolopez@gmail.com
SMTP_PASS=<TU_APP_PASSWORD>
MAIL_TO=josefranciscobarretolopez@gmail.com
MAIL_FROM=josefranciscobarretolopez@gmail.com

Notas

  • Activa la verificación en dos pasos en Gmail y genera una “Contraseña de aplicaciones” para usar como SMTP_PASS.
  • Usa el mismo correo en MAIL_FROM para evitar problemas de SPF/DMARC.
  • Después de configurar en Vercel, realiza un redeploy para que la función lea las variables.
